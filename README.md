# 🎨 EventConnect - Frontend (Angular)

![Angular](https://img.shields.io/badge/Angular-17-red)
![License](https://img.shields.io/badge/License-MIT-blue.svg)

## 📖 Description

This is the **frontend** for the **EventConnect** web application.  
It is built using **Angular** and provides a responsive interface for managing and participating in events.  
It communicates with the Spring Boot backend via REST APIs.

---

## 🚀 Features

- 🖥️ User-friendly dashboard
- 📋 Event browsing and reservation
- 📊 Event statistics (charts with Chart.js)
- 🏆 Rewards (discounts / free access display)
- 💳 Stripe payment integration
- 📝 Feedback submission for events


---
